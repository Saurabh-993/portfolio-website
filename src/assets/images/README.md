# Images

`src/components/ui/ProfileBadge.astro`:

- `profile-bw.jpg` — stylized / black-and-white portrait (default state)
- `profile-color.jpg` — real color portrait (hover state)

Currently both states use `profile-placeholder.svg` (the B&W state is
faked with a CSS `grayscale` filter).
